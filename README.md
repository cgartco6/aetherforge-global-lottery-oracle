# AETHERFORGE Global Lottery Oracle

Fully self-contained single-page website with agentic AI swarm + calculated risk prediction for **any major lottery on Earth**, including the newly added **South Africa Daily Lotto**.

## Features
- Select any lottery (SA Lotto, SA Powerball, SA Daily Lotto, US Powerball, Mega Millions, EuroMillions, UK Lotto, etc.)
- Real-time predicted numbers with full prize-tier risk matrix
- 1-year historical vault with realistic draws
- Live synthetic intelligence swarm (QI, SI, Agentic, Lottery Oracle, etc.)
- 100% offline — runs in any browser

## How to run (Windows 10 / old i5 laptop)
1. Download or clone this repository.
2. Double-click `index.html` — it opens directly in Chrome or Edge.
3. No installation, no API keys, no internet required after opening.

## Files
- `index.html` — The complete single-file application
- `lottery-data.json` — Optional external data file (embedded in HTML by default)
- `.gitignore` — Standard ignores for a static project
