# AETHERFORGE Global Lottery Oracle
Single-file HTML website with agentic AI swarm + calculated risk prediction for **every major lottery on Earth** (including brand-new SA Daily Lotto).

✅ Runs offline on any laptop (tested on old i5 Windows 10)  
✅ No API keys, no server  
✅ Just open index.html

Open in Chrome/Edge → select any lottery → get predicted numbers + exact risk matrix for every prize tier.
